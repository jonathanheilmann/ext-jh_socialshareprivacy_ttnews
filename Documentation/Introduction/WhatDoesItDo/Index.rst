What does it do?
^^^^^^^^^^^^^^^^

The jh\_socialshareprivacy\_ttnews extension includes the
socialshareprivacy plugin by heise.de to each tt\_news item.

Heise.de ducomentation (german):
`http://www.heise.de/extras/socialshareprivacy/
